# TestRotativa
Simple asp.net core app which use Rotativa.AspNetCore to generate a pdf.

# Usage
Clone, fork or download the repository. Open the solution in Visual Studio 2017 and run.<br/>
To view the pdf generated in browser use the /Pdf/Invoice url.<br/>
The code is located in the [PdfController class](https://github.com/albx/TestRotativa/blob/master/TestRotativa/Controllers/PdfController.cs).

The implementation of Rotativa is in the class library [TestRotativa.Acl](https://github.com/albx/TestRotativa/tree/master/TestRotativa.Acl)
